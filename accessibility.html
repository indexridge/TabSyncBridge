TabSyncBridge Accessibility

Effective date: May 5, 2026.

TabSyncBridge is designed to be usable with common iPhone and iPad accessibility features while keeping the app focused on local-network tab sync.

Supported Accessibility Features

  • VoiceOver: Main app screens use native iOS controls, readable labels, and specific labels for tab close, queued close, restore purchases, and linked-device removal actions.
  • Voice Control: Primary actions use visible button names or accessibility labels so they can be selected by voice.
  • Larger Text: Main screens use Dynamic Type text styles and scrolling layouts so larger text sizes can remain usable.
  • Sufficient Contrast: Main app screens use high-contrast text and control colours, including contrast-aware colour helpers for important text and actions.
  • Differentiate Without Color Alone: Connection, queued, and close states are described with text labels and not only by colour.
  • Dark Interface: Main app, onboarding, subscription, pairing, legal, and remote-tab screens adapt to iOS light and dark appearances.
  • Reduced Motion: The app does not require animation or motion to complete its main tasks.

Not Currently Claimed

  • Captions and Audio Descriptions: TabSyncBridge does not include audio or video content in its main app experience.

Device Support

The iPhone and iPad app use the same core interface patterns. Add iPad accessibility support in App Store Connect when Apple allows support to be published for the released iPad-capable app version.
