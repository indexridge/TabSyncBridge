TabSyncBridge

Local tab sync for Safari and desktop browsers.

View tab titles, keep linked devices tidy, and close selected remote tabs across your own devices on the same private local network.

iPhone app Manages linked devices, subscription access, and the shared remote tab list.
Safari extension Shares Safari tab titles and handles close requests when Safari can act on them.
Chromium extension Works with Chrome, Edge, Brave, Opera, Vivaldi, and compatible Chromium browsers.

Private by design

TabSyncBridge is built for local-network sync. It does not require a cloud account for the core sync loop, and it does not use third-party analytics, advertising SDKs, or tracking SDKs.

  • Local network only Pair and sync between devices you control on the same private Wi-Fi network.
  • Tab titles included See useful page titles instead of guessing from raw links.
  • Targeted close actions Close requests are routed to the browser profile that owns the selected tab.
  • Old profile cleanup Remove linked profiles that are no longer used.
