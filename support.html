TabSyncBridge Support

TabSyncBridge helps you view and close browser tabs across your own linked iPhone, Safari, Chromium-family browser, and desktop bridge setup.

App Store Add the Chromium companion Install guide

Requirements

  • iPhone with Local Network permission allowed.
  • Safari extension enabled if using Safari sync.
  • Chromium extension installed if using Chrome, Edge, Brave, Chromium, Opera, Vivaldi, or a compatible browser.
  • Desktop bridge installed when native messaging is required.
  • Devices connected to the same private Wi-Fi/local network.
  • Active subscription or free trial.

Basic Troubleshooting

  1. Make sure the iPhone and desktop are on the same Wi-Fi network.
  2. Open TabSyncBridge on the iPhone.
  3. Check that Local Network permission is allowed in iOS Settings.
  4. Reopen the Chromium extension.
  5. Remove old linked profiles if they no longer connect.
  6. Restart the browser if the extension was recently installed.
  7. Reinstall the desktop bridge if native messaging is not detected.

Frequently asked questions

How do I set up TabSyncBridge?

Install the TabSyncBridge app on your iPhone from the App Store. Enable the Safari extension for Safari workflows. For Chromium-family browsers, install the companion extension from the Chrome Web Store or Microsoft Edge Add-ons and the desktop bridge. Keep all devices on the same private Wi-Fi network and complete the pairing flow.

Why can't my devices find each other?

Make sure all devices are on the same Wi-Fi network. Check that Local Network permission is allowed for TabSyncBridge in iOS Settings. Restart the browser if the extension was recently installed. Reinstall the desktop bridge if native messaging is not detected.

How do I cancel my subscription or free trial?

On your iPhone, go to Settings → Apple Account → Subscriptions → TabSyncBridge, then choose Cancel Subscription. You can also open the App Store app, tap your profile, then Subscriptions.

How do I restore my purchase on a new device?

Open TabSyncBridge and use Restore Purchases on the subscription screen. Use the same Apple ID you used to start the trial or subscription.

Does TabSyncBridge work without Wi-Fi?

TabSyncBridge requires a private local network connection between your devices. Both devices must be on the same Wi-Fi network or local network for sync to work.

Which Chromium browsers are supported?

Chrome, Edge, Brave, Opera, Vivaldi, Chromium, and other compatible browsers that support Manifest V3 extensions.
