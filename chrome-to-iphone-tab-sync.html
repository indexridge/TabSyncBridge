TabSyncBridge

Chrome to iPhone tab sync without a cloud account.

TabSyncBridge links your iPhone app with the Chromium extension and desktop bridge so you can view desktop browser tab titles on your iPhone over your private local network.

View on the App Store Install the Chromium extension

How the Chromium workflow fits together

  1. iPhone app: manages subscription access, linked devices, and the shared remote tab list.
  2. Chromium extension: reads eligible tab titles and sends selected close-tab requests.
  3. Desktop bridge: enables native messaging and local relay for Chromium-family browser workflows.

Supported browser families

Use the companion extension with supported Chromium-family browsers such as Chrome, Edge, Brave, Opera, Vivaldi, Chromium, and compatible browsers that support Manifest V3 extensions.

Sync is designed for your own devices on the same private local network. TabSyncBridge does not require a cloud account for the core sync loop.
