TabSyncBridge

Close selected remote tabs from the right browser profile.

TabSyncBridge routes close-tab requests to the browser profile that owns the selected tab, so remote tab control stays tied to your own linked devices.

View on the App Store Add the Chromium companion

Remote tab control, scoped locally

  • Owner-aware close requests The close request targets the origin that owns the selected tab.
  • Tab title context Readable titles make it easier to choose the right remote tab.
  • Profile cleanup Remove old linked profiles that are no longer part of your workflow.
  • Private network workflow Core sync and remote tab control are designed for your own local network.

Designed for your own devices

Use TabSyncBridge only with devices and browser profiles you own or have permission to control. It is not a cloud account, shared surveillance tool, or third-party tab relay.

Close actions depend on the linked browser and extension being available to receive the request.
