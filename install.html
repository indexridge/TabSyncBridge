Why install

TabSyncBridge helps you view tab titles, reconnect to useful pages, clean up old linked profiles, and send selected close-tab requests to the browser profile that owns the tab. The core sync loop is designed for your own devices on the same private local network.

  • Safari plus Chrome Keep iPhone Safari and desktop Chromium tabs in one local workflow without moving everything into one browser ecosystem.
  • No cloud account Pair your own devices on the same private Wi-Fi network instead of creating a new sync account or relaying tab data through a developer server.
  • Paid value is control The iPhone app unlocks the shared remote tab list, linked-device cleanup, and close-tab requests after the App Store trial starts.

TabSyncBridge is £2.99 per month after a 3-month free trial. Purchases, renewals, cancellations, refunds, and restore purchases are handled by Apple through the App Store.

After you install, open the iPhone app on the same Wi-Fi as the Mac or PC you want to link so pairing can complete. If something blocks access after the trial or you need to cancel or restore billing, use the Support page first; it explains Apple subscription settings and restore purchases in one place.
